Conference Agenda Parser

A Python program that parses conference agendas, categorizing entries and generating structured outputs.

Motivation

The Conference Agenda Parser was created to simplify the process of organizing and categorizing information from conference agendas. Attending conferences often generates a wealth of information, and this tool aims to streamline the task of extracting relevant details, making it easier for attendees to plan and prepare for sessions.

Implementation

This project is implemented in Python, utilizing the following tools and libraries:

  • Python: The primary programming language used for development.
  • PrettyTable: A library for generating formatted tables for easy readability of output data.
  • Pandas: Utilized for data manipulation and exporting structured information into Excel spreadsheets.

The script processes input text files containing conference agendas, categorizes the details, and outputs the data in both table format and Excel. This approach not only streamlines the workflow but also provides users with organized access to essential event information.

Notably, I have ensured that the code is well-documented, providing clear explanations and usage instructions, which not only aids in maintenance but also demonstrates my commitment to best practices in software development.
