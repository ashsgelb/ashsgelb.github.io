Upper to Lower Case Converter

A Python script that converts text from uppercase to lowercase in Excel documents while preserving exceptions.

Motivation

The Upper to Lower Case Converter was developed to streamline the process of text formatting in Excel documents, particularly when dealing with large datasets. Manually converting text can be tedious and prone to errors, especially when exceptions like acronyms and proper nouns need to be preserved. This script automates the task, enhancing productivity and ensuring consistency.

Implementation

The converter is implemented in Python, key features include:

  • Automation: The script reads specified Excel files, converts all text to lowercase while preserving designated exceptions.
  • Exception Handling: Supports the preservation of acronyms and proper nouns through a customizable list, preventing unintended formatting changes.
  • Error Handing: Includes robust error handling to manage file access issues and ensure smooth execution.
